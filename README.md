# 📊 Telecom Customer Churn Analysis & Prediction System

## 📌 Project Overview

The Telecom Customer Churn Analysis & Prediction System is a Machine Learning and Data Analytics project developed to analyze customer behavior and predict whether a telecom customer is likely to leave the company (churn) or stay.

This project combines Data Analysis, Data Visualization, Machine Learning, and Web Application Development to help telecom companies improve customer retention strategies.
